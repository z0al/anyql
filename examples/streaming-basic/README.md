# Basic Streaming example

[![Open in CodeSandbox][csb]][box]

## Running locally

```sh
git clone https://github.com/z0al/oktane.git

cd oktane/examples/streaming-basic

npm install # or yarn

npm start # or yarn start
```

Visit https://localhost:3000 on your browser.
